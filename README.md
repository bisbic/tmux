# README

## Setup
Clone this repo:
```
git clone https://github.com/bisbic/tmux.git ~/.config/tmux/
```
Install tmux package manager
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
Load config:
```
tmux source ~/.config/tmux/tmux.conf
```
Install plugins:
```
Leader I
```
